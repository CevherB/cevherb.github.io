Wichtige CSS-Befehle

Farbe ändern

Mit dem Befehl "color" kannst du die Farbe des Textes ändern. Zum Beispiel:

h1 {
  color: #007AFF;
}

Schriftgröße ändern

Mit dem Befehl "font-size" kannst du die Schriftgröße des Textes ändern. Zum Beispiel:

h1 {
  font-size: 36px;
}

Schriftart ändern

Mit dem Befehl "font-family" kannst du die Schriftart des Textes ändern. Zum Beispiel:

    h1 {
      font-family: Arial, sans-serif;
    }

Schriftgröße ändern

Mit dem Befehl "font-size" kannst du die Schriftgröße des Textes ändern. Zum Beispiel:

h1 {
  font-size: 36px;
}

Schriftstil ändern

Mit dem Befehl "font-weight" kannst du den Schriftstil des Textes ändern. Zum Beispiel:

h1 {
  font-weight: bold;
}

Hintergrundfarbe hinzufügen

Mit dem Befehl "background-color" kannst du einer Box eine Hintergrundfarbe hinzufügen. Zum Beispiel:

.box {
  background-color: #E9EBEE;
  padding: 20px;
}

Hintergrundbild hinzufügen

Mit dem Befehl "background-image" kannst du einer Box ein Hintergrundbild hinzufügen. Zum Beispiel:

.box {
  background-image: url("bg.jpg");
  padding: 20px;
  background-size: cover;
}

Transparenz hinzufügen

Mit dem Befehl "opacity" kannst du einer Box eine Transparenz hinzufügen. Zum Beispiel:

.box {
  background-color: rgba(233, 235, 238, 0.8);
  padding: 20px;
}

Text-Schatten hinzufügen

Mit dem Befehl "text-shadow" kannst du einem Text einen Schatten hinzufügen. Zum Beispiel:

h1 {
  text-shadow: 2px 2px #333;
}

Box-Schatten hinzufügen

Mit dem Befehl "box-shadow" kannst du einer Box einen Schatten hinzufügen. Zum Beispiel:

.box {
  box-shadow: 2px 2px 5px #333;
  padding: 20px;
}

Runde Ecken hinzufügen

Mit dem Befehl "border-radius" kannst du einer Box runde Ecken hinzufügen. Zum Beispiel:

.box {
  border-radius: 10px;
  padding: 20px;
}

Margin hinzufügen

Mit dem Befehl "margin" kannst du einer Box einen Abstand zu den umgebenden Elementen hinzufügen. Zum Beispiel:

.box {
  background-color: #E9EBEE;
  padding: 20px;
  margin: 40px;
}

Padding hinzufügen

Mit dem Befehl "padding" kannst du einer Box einen Innenabstand hinzufügen. Zum Beispiel:

.box {
  background-color: #E9EBEE;
  padding: 20px;
  margin: 40px;
}
